# mficco.com - Personal Website

## Overview
A simple static personal website with pages for Home, Writing, Books, and About.

## Project Structure
- `index.html` - Home page
- `writing.html` - Writing page
- `books.html` - Books page  
- `about.html` - About page
- `projects.html` - Projects page
- `style.css` - Shared stylesheet
- `CNAME` - Custom domain configuration

## Technology
- Pure HTML/CSS static website
- Google Fonts (Roboto Mono)
- No build system required

## Running Locally
The site is served using Python's built-in HTTP server on port 5000.

## Deployment
Configured as a static deployment serving files from the root directory.
